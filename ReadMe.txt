우리팀을 위한 초간단 가이드

* 프로그램 실행시 기본 바디모델 생성 -> 해당 오른쪽 왼쪽 바퀴가 지면에 닿는 제일 아래부분임

* 타이틀의 숫자들은 [헤드] [바디] [오브젝트] 타입별 갯수

* create option
	트랜스폼 설정, 타입, 컬러 선택후 Add Cube 생성

* type
	 head , body , object 선택시 저장할때 _head.bin, _body.bin, _object.bin
	자동으로 붙어서 저장됨. 타입을 지정해서 render옵션으로 해당 부위만 볼수있음
	타입을 선택해야 해당하는 리스트로 저장이됨 부위선택 필수

* selected cube info
	scale, angle, pos 세부조정
	선택된 오브젝트를 방향키로 x,z 위치 조절 , PageUp , PageDown으로 y값 조절 
	조절되는 값은 0.1 단위

* Object List
	생성된 오브젝트 리스트
	마우스 클릭시 해당 오브젝트가 현재선택된 오브젝트로 설정됨

* 카메라
	카메라 이동	 	WSAD
	마우스휠 		ZOOM IN/OUT
	오브젝트 회전 	QERF
	정면, 후면, 오른쪽 , 왼쪽 , 위,아래  보기  F1~F (누르고 있을때만)
	스페이스바		카메라 초기위치로 이동 (0, 10, -20)위치

* 선택된 오브젝트가 있을경우 Delete키를 누르면 해당 오브젝트를 삭제함 (혹은 Delete Cube 버튼)

* 선택된 큐브가 있을때 다른 큐브를 마우스 우클릭하게 되면 색상을 카피함
	- 혹은 우측상단의 색상표를 선택하고 다른 큐브를 마우스 우클릭하면 색상을 적용함

* 선택된 큐브가 있을때 다른 큐브를 마우스위에 올려두고 F12를 누르면 스케일,회전값을 카피함 (위치는 X)

* Ctrl + s로 혹은 좌상단의 save as file로 저장 가능 , 해당 매쉬의 이름만 지정하면 자동으로 [타입].bin 붙어서 저장됨
* 로드기능은 어떤 타입으로 로드할지 선택하는 옵션임 타입변경이 불가능하니 잘 선택해줘야함
